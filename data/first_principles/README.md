# First Principles of Science
### A Comprehensive Research Project

> *"If you can't explain it simply, you don't understand it well enough."* — Albert Einstein

---

## Project Overview

This project is a systematic investigation into the **first principles** — the foundational axioms, laws, postulates, and irreducible truths — of every major branch of science. The goal is to trace each scientific discipline back to its most fundamental assumptions and build a complete map of the bedrock upon which all scientific knowledge rests.

**What are First Principles?**
First principles are the most basic propositions or assumptions that cannot be deduced from any other proposition or assumption within a given domain. They are the starting points from which all other knowledge in a field is derived. In Aristotle's formulation: *"A first principle is the first basis from which a thing is known."*

## Methodology

Each branch is investigated following a consistent protocol:

1. **Define the domain** and its boundaries
2. **Identify axioms/postulates** — starting assumptions that cannot be derived from simpler elements
3. **Identify fundamental laws** — core laws from which all other results derive
4. **Trace dependencies** — map which principles depend on other fields
5. **Classify each principle** using a standardized taxonomy:

| Tag | Meaning |
|-----|---------|
| `AXIOM` | Assumed true without proof; foundational by definition |
| `LAW` | Empirically established fundamental law |
| `POSTULATE` | Assumed for theory construction |
| `PRINCIPLE` | Guiding truth derived from axioms/laws |
| `THEOREM` | Provable but so fundamental it serves as a building block |
| `CONVENTION` | Definitional choice shaping the framework |
